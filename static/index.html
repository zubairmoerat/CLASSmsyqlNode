A table containing HTTP Methods

Path HTTP Methods Description
/ GET Renders an index.html
/add POST Submit data to be processed to a specified resource.
/update PATCH Apply partial modifications to a resource.
/delete DELETE Removal of a resource identified by the URL.
/replace PUT Replace an existing resource
